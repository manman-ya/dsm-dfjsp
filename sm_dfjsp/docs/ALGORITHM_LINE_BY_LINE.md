## 1) `src/smdfjsp/eda_ts/algorithm.py`

### 1.1 文件级说明

这是主算法 EDA-TS 的完整实现，包含：

- 概率模型初始化（PMA/PMS/PMM）
- 采样与启发式构造
- 精英更新
- 三种邻域 + 禁忌搜索
- 非支配记忆池
- 迭代主循环

### 1.2 行号注释

- 1-21：导入依赖。`encoding` 负责编码修复，`pareto` 负责多目标排序，`evaluate_individual` 是统一评价器。
- 24-39：`EDATSConfig` 参数类。`alpha/beta/gamma` 是 3 个概率矩阵学习率，`mu` 是精英比例，`epsilon` 是禁忌惩罚强度。
- 42-45：`RunResult`，输出非支配解和迭代历史。
- 48：定义 `EDATS` 主类。
- 49-59：构造函数。把实例映射、索引、兼容集合准备好，然后初始化概率矩阵。
- 61-74：初始化 PMA（作业 -> SRU 概率），默认均匀分布。
- 75-83：初始化 PMS（作业 -> OS 位置概率），按类型维度建立长度 `kx`。
- 84-96：初始化 PMM（工序 -> 机器概率），仅对可行 `(job,op,sru)` 建概率向量。
- 98-105：`_sample_ua`，按 PMA 采样每个作业分配 SRU。
- 107-115：`_md_ua`，启发式分配（选运输时间最短 SRU）。
- 117-156：`_sample_os`，按 PMS 给每个作业令牌选择位置；最后修补 `-1` 空位。
- 158-173：`_sample_ms`，按 PMM 采样机器；若没有 PMM 信息则退回随机可行机器。
- 175-185：`_mc_ms`，机器选取准则为最小 `pt * cp`（加工成本导向）。
- 187-208：`_mct_ms`，机器选取准则为最小预计完成时间（时间导向）。
- 210-230：`_build_individual`，构建一个完整个体；多群体模式下混合 3 种 MS 构造策略。
- 232-237：`_evaluate_population`，批量调用评价器回填目标值。
- 239-252：`_select_elite`，非支配排序 + 拥挤距离选精英。
- 254-268：更新 PMA，统计精英中 UA 频率并指数平滑更新。
- 270-287：更新 PMS，统计精英中作业令牌位置分布并平滑更新。
- 288-308：更新 PMM，统计精英中机器选择频率并平滑更新。
- 310-331：邻域 I，作业在同类型可行 SRU 间重分配。
- 333-357：邻域 II，OS 做插入扰动（位置移动）。
- 359-382：邻域 III，MS 改机（在可行机器集合内替换）。
- 384-385：`_penalized`，对目标添加频率惩罚避免重复动作。
- 387-393：`_tabu_search` 初始化当前解与最优解。
- 394-397：定义禁忌表和两类记忆频率（SRU分配频率、机器动作频率）。
- 399-407：每轮生成 3 种邻域并评估。
- 408-422：对邻域解计算惩罚后目标并排序。
- 423-433：禁忌准则选候选；若都禁忌则选最优惩罚解。
- 435-441：更新动作频率记忆。
- 443-446：若当前解支配历史最优则替换。
- 449-452：`run()` 初始化种群并评价。
- 453-459：若开启记忆池，则把初代非支配解放入 `nd_pool`。
- 462-464：主循环：迭代上限或时间上限停止。
- 465-468：选精英并更新概率矩阵。
- 470-471：按新概率重采样一代。
- 473-481：若启用 TS，则从记忆池抽种子做局部强化并回填到新种群。
- 483-496：环境选择：非支配排序 + 拥挤距离，截断回 `popsize`。
- 498-509：更新 `nd_pool`（开启/关闭记忆两种分支）。
- 510-512：记录本代最优成本、最优完工期、记忆池大小。
- 514：返回结果。

## 11) 如何用这份逐行注释学习

推荐你按下面顺序：

1. `core/encoding.py`（先理解“解怎么表达”）
2. `model/evaluator.py`（再理解“解怎么打分”）
3. `eda_ts/algorithm.py`（主算法）
4. `baselines` 三类算法（对照主算法）
5. `model/gurobi_model.py`（精确模型对照）

每看完一个文件，去跑一次：

```powershell
$env:PYTHONPATH='src'
python scripts/run_smoke.py
```

这样“代码理解”和“运行结果”会绑定在一起，学得最快。
